RespirAlert SC

Plataforma Inteligente de Monitoramento Climático e Hospitalar para Apoio à Prevenção de Doenças Respiratórias em Santa Catarina.
Dados públicos. Análise real. Impacto social.
Por que este projeto importa?
"Este projeto transforma dados públicos em conhecimento acessível para apoiar estudos acadêmicos, pesquisas e ações preventivas. A plataforma demonstra como a análise de dados pode auxiliar na compreensão de padrões relacionados à saúde pública e às condições climáticas, fortalecendo o uso de evidências na tomada de decisão."

Santa Catarina concentra municípios com temperaturas médias históricas abaixo de 12°C (São Joaquim, Urupema, Bom Jardim da Serra), associadas a períodos de alta incidência de doenças respiratórias. A análise sistemática da relação entre clima e saúde, utilizando dados públicos do DATASUS e INMET, permite identificar padrões sazonais e municípios de maior vulnerabilidade — informação essencial para planejamento preventivo em saúde pública.

Diferenciais Técnicos
✦
Pipeline resiliente em 3 etapas: API oficial → download público → último dataset local válido, sempre com status registrado.
✦
Information Value adaptado: técnica de scorecard de crédito aplicada à saúde pública para medir poder preditivo da temperatura.
✦
Insights automáticos: interpretações baseadas nos indicadores estatísticos de cada município.
✦
Mapa com Canvas API: renderização de GeoJSON municipal via canvas sem bibliotecas externas.
✦
Exportação PDF client-side: jsPDF + html2canvas carregados sob demanda, sem backend.
✦
Zero banco de dados, zero backend: 100% estático — JSON + CSV + GitHub Pages.

⚠️ Segurança Jurídica e Metodológica: Este sistema utiliza dados públicos oficiais para fins acadêmicos, científicos e informativos. Os resultados representam associações estatísticas e não estabelecem causalidade. Este sistema não realiza diagnóstico médico e não substitui orientação médica, epidemiológica ou decisões governamentais. Eventuais inconsistências podem decorrer das limitações das bases públicas utilizadas.
